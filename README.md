# Como operar e visualizar?

O arquivo que utiliza as implementações modulares e imprime no terminal do usuário os resultados de fato, é o arquivo *petr4_spline.py*. Acesse-o e o rode. Se todos os arquivos estiverem no mesmo diretório, é esperado que o código opere normalmente.

O código não prevê robustez quanto a eventuais atualizações ou mudanças de funcionamento de bibliotecas auxiliares ou do próprio Python.
